Lint any Obsidian plugin for compliance — right from your browser.

Live demo of oplint. Same engine runs in CI via oplint-action — lint on push, badge, PR comment, HTML artifact.

⏳ Heads up — first run may be slow. Hosted on Render's free plan ❤️ (0.1 CPU · 512 MB), which spins down after inactivity — cold start can take 30–60 s. Each run also clones the plugin repository before linting it, adding network time. Neither reflects real lint performance — oplint lints a typical plugin in 500 ms – 5 s locally or in CI.
